📊 Superstore_Analysis Analysis using SQL

🧾 Project Description

This project is a complete SQL-based Sales Analytics System designed to simulate a real-world retail/e-commerce business database.

The project includes a fully structured relational database, sample data for 3 years of sales, and 20 business analysis SQL queries used to generate meaningful insights such as revenue, profit, customer behavior, product performance, and sales trends.

This project is ideal for SQL beginners, students, and data analyst portfolios.

🎯 Objectives

The main goals of this project are:

Design a real-world relational database schema

Create sample business data for analysis

Perform data analysis using SQL queries

Generate business insights from raw sales data

Demonstrate SQL skills for portfolio / resume

🏢 Database Overview

The database represents a SuperStore / E-Commerce company and contains 15 normalized tables.
